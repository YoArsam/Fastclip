# FastClip

MVP scaffold for the Fast Clip iPhone app flow, built with Expo + React Navigation.

## Run locally

1. Install dependencies:

   npm install

2. Start (iOS / Web):

   npm run ios
   npm run web

## Deploy to Vercel (Web preview)

Vercel deploys the Expo **web** build (not the iOS app runtime).

Build command:

  npm run build

Output directory:

  dist
